# Anthill Spider

Anthill Spider is a browser extension that extracts ChatGPT conversations from
the web UI and exports them as structured JSON.

### Output
- `chatgpt_conversations_export.json`

### Responsibilities
- Discover conversations
- Load each conversation page
- Extract rendered message content
- Preserve message order and roles

Spider performs **no cleaning, filtering, or transformation**.
